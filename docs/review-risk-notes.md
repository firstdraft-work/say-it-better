# 微信审核风险提示

## 1. 录音能力

风险点：

- 未说明录音用途
- 没有隐私说明

建议：

- 在页面或隐私政策中明确说明“录音仅用于语音转文字和表达优化”

## 2. AI 内容生成

风险点：

- 审核人员不清楚产品用途
- 误以为是泛内容生成工具

建议：

- 强调产品是“沟通表达优化工具”
- 强调场景是“把原话变得更得体”

## 3. 敏感内容

风险点：

- 用户可能输入攻击性、违规或敏感内容

建议：

- 在产品说明里强调仅用于合法合规沟通场景
- 后续可补内容安全审核

## 4. 体验一致性

风险点：

- 真机请求慢
- 语音链路偶发失败

建议：

- 提审前至少做一次完整真机回归
- 若生成速度慢，给出明确 loading 提示
