# 隐私说明文案草案

## 录音权限用途说明

用于将你录制的语音内容转换为文字，并进一步生成更合适的沟通表达方式。

## 隐私政策中的数据说明建议

### 我们会收集什么

- 你输入的文本内容
- 你主动录制并上传的语音内容
- 系统生成的优化表达结果
- 你主动触发的收藏、复制、播放等使用行为

### 我们为什么收集这些信息

- 文本内容：用于生成更得体、更符合场景的人际沟通表达
- 语音内容：用于语音转文字
- 生成结果：用于展示历史记录，便于你后续查看和复用
- 使用行为：用于优化产品体验

### 数据处理方式

- 文本优化由云端大模型完成
- 语音识别由腾讯云语音识别服务完成
- 语音合成由腾讯云语音合成服务完成

### 用户可见说明建议

- 录音内容仅用于语音转文字与表达优化
- 你可随时删除历史记录
- 请不要输入或录制违法违规、侵犯他人隐私的内容
