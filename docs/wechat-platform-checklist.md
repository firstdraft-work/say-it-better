# 微信小程序上线前微信侧配置清单

## 1. 小程序主体与 AppID

- 申请正式小程序主体
- 拿到正式 `AppID`
- 把 `miniprogram/project.config.json` 里的 `touristappid` 换成真实 `AppID`

## 2. 开发者与成员

- 配置项目成员
- 配置体验成员
- 确认谁有上传、发布、查看数据权限

## 3. 合法域名

这个项目至少要配置以下域名，通常都填同一个 API 域名：

- `request 合法域名`
  - `https://api.example.com`
- `uploadFile 合法域名`
  - `https://api.example.com`
- `downloadFile 合法域名`
  - `https://api.example.com`

当前项目不依赖 WebSocket，可先不配 `socket`。

## 4. HTTPS

- 域名必须可公网访问
- 必须是 `HTTPS`
- 证书要有效

## 5. 隐私与权限说明

这个项目涉及：

- 录音
- 文本输入
- 云端处理文本和语音

所以需要补：

- 隐私政策链接
- 录音用途说明
- 数据处理说明

## 6. 类目与审核材料

建议提前准备：

- 产品功能说明
- 录音和 AI 处理用途说明
- 是否生成用户可复制文本
- 是否生成语音播放内容

## 7. 体验版与审核

- 上传代码
- 生成体验版
- 邀请体验成员验证
- 提交审核
- 审核通过后发布

## 8. 这个项目实际上线前还需要确认

- API 域名已从本地地址切到公网域名
- 后端已部署在阿里云轻量服务器
- MySQL、GLM、腾讯云 ASR、腾讯云 TTS 的环境变量已在服务器配置
- `env.js` 已切到线上域名
