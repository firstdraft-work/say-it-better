# 人物关系与沟通边界矩阵

## 核心原则

- 不一味“更客气”，而是“更得体”
- 保留原意，不改变立场，不编造事实
- 人物关系决定礼貌层级，沟通目标决定语气力度
- 拒绝和边界表达允许坚定，不能软到失真

## 关系边界

### 对领导

- 关键词：尊重、克制、清晰、有回旋空间
- 不该出现：命令感、阴阳怪气、质问感
- 可以保留：不同意见、拒绝、风险提醒
- 推荐语气：礼貌但不讨好

### 对下属

- 关键词：明确、专业、边界清楚
- 不该出现：过度客气导致失去权威
- 可以保留：时间要求、责任要求、反馈要求
- 推荐语气：直接但不压人

### 对同事

- 关键词：平等、自然、协作感
- 不该出现：卑微、官腔、压迫式命令
- 可以保留：催进度、拒绝、边界
- 推荐语气：不卑不亢

### 对长辈

- 关键词：礼貌、缓冲、留余地
- 不该出现：生硬反驳、明显冒犯
- 可以保留：个人边界、委婉拒绝
- 推荐语气：柔和但不失立场

### 对伴侣

- 关键词：自然口语、有情绪照顾
- 不该出现：像汇报工作、像客服
- 可以保留：表达不满、提出需求、划边界
- 推荐语气：真实、温和、有人味

### 对朋友

- 关键词：轻松、自然、不过度正式
- 不该出现：公文腔
- 可以保留：拒绝、提醒、解释
- 推荐语气：口语化

## 目标边界

### 请求

- 重点：说清楚需求，不要模糊到对方不知道要做什么

### 拒绝

- 重点：拒绝必须明确，不能表面礼貌但实质暧昧

### 提醒

- 重点：要有推进力，但避免质问感

### 解释

- 重点：说明原因，不要过度防御

### 道歉

- 重点：真诚负责，不用无限下沉姿态

### 表达边界

- 重点：允许坚定，不能只剩“求理解”

## 三档输出原则

- 非常委婉：更柔和，更照顾对方情绪，但不丢核心意思
- 平衡推荐：最符合场景和关系的版本
- 直接但礼貌：保留边界和清晰度，减少攻击性
